# Prompts para Google Stitch

## Prompt base

```text
Use the attached DESIGN.md as the design system.
Create a responsive web app interface for Nomadfoot Manager, a lightweight fictional football management game.
The UI should feel like a compact modern dashboard for a classic manager game: fast, readable, tactical and data-driven.
Do not create a marketing landing page.
Do not use real football clubs, real players, protected logos or copied game assets.
Use fictional club identity, compact tables, clear tabs and a visible primary action.
```

## Tela inicial

```text
Design the start screen for Nomadfoot Manager.
Include brand mark, game title, primary action "Novo jogo", secondary action "Carregar jogo salvo", and a subtle tactical football field visual.
Keep the screen direct and playable, not promotional.
```

## Escolha de clube

```text
Design the club selection screen.
Show a responsive grid of fictional club cards with crest, club name, league, reputation, squad strength and budget.
Each card needs one clear action: "Treinar clube".
Keep cards compact and scannable.
```

## Dashboard da carreira

```text
Design the main career dashboard.
Include club header, season, tabs, club strength, table position, budget, next match, latest news, top 5 standings and recent results.
The primary action must be "Avançar rodada".
Use dense but calm dashboard layout.
```

## Elenco

```text
Design the squad screen.
Use a readable table with position, player, age, quality, value, salary, fitness and status.
Make tired players visible without creating noisy colors.
Prioritize desktop and mobile readability.
```

## Calendário e tabela

```text
Design calendar and league table screens for a football manager.
Calendar should show rounds and match rows.
League table should show position, club, points, played, wins, draws, losses and goal difference.
Highlight the user's club clearly.
```
