# Memória longa do projeto

Esta pasta guarda histórico, conteúdo preservado e registros de baixa frequência do projeto Nomadfoot.

Use os arquivos curtos em `docs/` para contexto cotidiano. Consulte esta memória longa quando precisar auditar decisões, recuperar versões anteriores, entender histórico ou verificar detalhes que foram retirados da leitura frequente.

## Arquivos principais

- [_index.md](./_index.md): mapa de consulta da memória longa.
- [_migration-log.md](./_migration-log.md): registro das migrações e preservações realizadas.
- [snapshots/](./snapshots/): cópias literais de arquivos antes de otimizações.
