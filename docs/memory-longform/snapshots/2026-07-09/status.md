# Status

Data: 2026-07-09

## Estado atual

Base inicial criada a partir do PRD. O projeto já possui uma versão jogável no navegador com novo jogo, escolha de clube, elenco, calendário, simulação de rodadas, tabela, fim de temporada, nova temporada simples e save/load local.

## Escopo entregue

- Estrutura Vite com JavaScript puro.
- Dados fictícios de 8 clubes e elencos gerados localmente.
- Liga Livre em pontos corridos com turno e returno.
- Simulação simples de partidas.
- Interface funcional responsiva.
- Persistência via `localStorage`.
- Documentação inicial de mecânicas.

## Fora desta entrega

- Transferências.
- Escalação manual.
- Lesões e suspensões.
- Estatísticas individuais.
- Backend, login ou multiplayer.
- Balanceamento fino de resultados.

## Próximo marco sugerido

Adicionar gestão básica: escalação inicial, impacto mais claro de condição física e primeira versão de mercado de transferências.
