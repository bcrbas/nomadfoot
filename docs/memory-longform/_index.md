# Índice da memória longa

## Finalidade

Mapa de consulta para os arquivos de memória longa do projeto.

Use este índice quando precisar recuperar histórico, decisões antigas, registros completos, versões anteriores de status, tarefas concluídas, detalhes de mecânicas ou justificativas que não precisam ser lidas em toda tarefa cotidiana.

O `prd.md` atual não cita arquivos específicos de brands, assets, meetings, copy, offer ou research. Se esses temas forem criados no futuro, eles devem ganhar arquivo curto em `docs/` e longform correspondente nesta pasta.
