# Status

## Função deste arquivo

Contexto operacional do estado atual do projeto. Para histórico completo e versões anteriores, consulte [status-history.md](./memory-longform/status/status-history.md).

## Estado atual

Data: 2026-07-09

Base inicial jogável criada a partir do PRD. O navegador já permite novo jogo, escolha de clube, elenco, calendário, simulação de rodadas, tabela, fim de temporada, nova temporada simples e save/load local.

## Escopo vigente

- Estrutura Vite com JavaScript puro.
- Dados fictícios de 8 clubes e elencos gerados localmente.
- Liga Livre em pontos corridos com turno e returno.
- Simulação simples de partidas.
- Interface funcional responsiva.
- Persistência via `localStorage`.
- Documentação inicial de mecânicas, status, decisões e tarefas.

## Fora do MVP atual

- Transferências, escalação manual, lesões, suspensões e estatísticas individuais.
- Backend, login, rankings globais ou multiplayer.
- Balanceamento fino da simulação.

## Próximo marco

Adicionar gestão básica: escalação inicial, impacto mais claro de condição física e primeira versão de mercado de transferências.
